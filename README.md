# 个人笔记-markdown

一个专注浏览的个人笔记.

### 快速使用

```bash
git clone https://github.com/hanxi/note-md.git
cd node-md
npm install
npm start
```

- 笔记存储在 `note` 目录
- 浏览器访问 <http://localhost:3000>
